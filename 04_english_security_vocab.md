# English Security Vocab (starter)

## Core terms
- Vulnerability
- Exploit
- Proof of Concept (PoC)
- Impact
- Remediation
- Unauthorized access
- Access control
- Session
- Input validation
- Output encoding

## Useful report phrases
- The application fails to enforce proper access control.
- An authenticated user can access another user's data.
- This issue can be reproduced consistently.
- The vulnerability may lead to unauthorized data disclosure.
- I recommend implementing server-side authorization checks.

## Weekly micro-goal
- Выучить 5 новых терминов.
- Использовать минимум 3 фразы в учебном репорте.
